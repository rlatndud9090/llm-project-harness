# UI Engineer 어댑터

공용 기준은 `docs/harness/roles/ui-engineer.md`다.

필수:

- 첫 화면은 실제 일일 퀴즈 사용 표면이어야 한다.
- UI는 domain state를 렌더링하고 domain rule을 직접 계산하지 않는다.
- 모바일/데스크톱에서 텍스트와 컨트롤 겹침을 확인한다.
- 결과 공유에는 턴 수가 포함되어야 한다.
